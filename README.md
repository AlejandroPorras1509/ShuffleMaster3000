# ShuffleMaster3000
Java project that simulates a customizable deck of cards, supports shuffling, dealing hands, and generating a histogram based on 100,000 simulations. Demonstrates arrays, classes, randomness, and statistical analysis through a console-based interface.

## Project Description
Java application that simulates a customizable deck of cards. It supports shuffling, dealing hands, and generating a histogram from 100,000 simulations. The project demonstrates the use of arrays, classes, randomness, and basic statistical analysis through a console-based interface.

## Technologies Used
- Java 17+
- Object-Oriented Programming (OOP)
- Arrays
- Randomization
- Console Input/Output
